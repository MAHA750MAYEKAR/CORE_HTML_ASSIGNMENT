1. HTML

    Primary use case Benefits
    Building structure of web page
    • HTML allows you to browse the internet with ease.
    • HTML allows you to create web documents.

  2. CSS

    Primary use case Benefits
    CSS is used for defining the styles for web pages. It describes the look and formatting of a document which is written in a markup language
    • Separation of Content and Style
    • CSS enables consistent styling across an entire website or multiple pages.
    • External CSS files can be cached by web browsers, allowing them to load faster
    • Responsive Design

  3. JavaScript

    Primary use case Benefits
    JavaScript is commonly used to make web pages dynamic. As a client scripting language, it provides websites with interactivity
    • Reduces Load on the Server
    • The versatility of JavaScript is its most valuable quality. There are many different ways to integrate it into your website
    • JavaScript's ability to run on the client-side helps reduce server load and overhead
    • JavaScript can be interpreted differently by various browsers, making it necessary to test and run the code on different platforms before publishing.
